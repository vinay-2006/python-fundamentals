## [1.0.0] - 2025-12-04

### Added
- **Initial release of the Python Utility Toolkit.**
- Implemented 16 core utility functions covering Math, Strings, and Data Processing:
  - prime checking  
  - factorial  
  - palindrome checks  
  - dictionary merging  
  - statistics calculation  
  - list flattening  
  - unique element extraction  
  - safe literal parsing  
- Added robust input validation (type & value checks) across all functions.
- Included interactive Command Line Interface (`if __name__ == "__main__":`) for testing utilities.
- Added `save_history` utility to log CLI interactions to `utility_history.jsonl`.
- Created foundational project files:
  - `README.md`
  - `CONTRIBUTING.md`
  - `LICENSE`
  - `.gitignore`
- Added initial unit tests in `tests/test_utility_toolkit.py`.

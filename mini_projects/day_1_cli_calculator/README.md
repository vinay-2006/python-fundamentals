# Day 1 — CLI Calculator (Mini Project)

A command-line calculator that supports multiple arithmetic operations and stores a calculation history.

## Features
> Supports +, -, *, /, //, %, **  
> Input validation  
> Error handling for invalid numbers and operations  
> Saves past calculations in `history.txt`

## Files
> calculator.py — Main program  
> history.txt — Sample history output  

## How to Run
python mini_projects/day_1_cli_calculator/calculator.py

## Notes
This mini project is part of Day 1 of Python Fundamentals.
